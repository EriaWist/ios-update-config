PocketGit 隱私權政策

最後更新日期：2026 年 6 月 10 日

一、總覽

PocketGit 是一款為 iPhone 設計的 Git repository 管理工具。 我們重視使用者的隱私權。本隱私權政策說明 PocketGit 如何處理資料，以及使用者在使用本 App 時需要了解的隱私權資訊。

PocketGit 不會主動收集、追蹤、販售或分享使用者的個人資料。

二、我們收集的資料

PocketGit 本身不會將使用者的個人資料傳送到我們的伺服器，也不會使用廣告追蹤或第三方分析工具。

App 內產生或使用的資料，例如 Git repository、commit 資訊、branch 資訊、remote 設定、App 設定與相關檔案，主要儲存在使用者的裝置本機。

三、本機儲存資料

為了提供 Git repository 管理功能，PocketGit 可能會在使用者裝置本機儲存以下資料：

  • 使用者加入或建立的 Git repository
  • repository 的檔案、commit、branch、remote 等 Git 相關資訊
  • App 偏好設定與操作狀態
  • 使用者自行輸入或設定的 Git 遠端連線資訊
  • 使用者自行儲存於裝置中的憑證或 access token

這些資料主要用於 App 功能運作，不會由 PocketGit 主動上傳至我們的伺服器。

四、第三方 Git 服務

使用者可以選擇將 PocketGit 與 GitHub 或其他第三方 Git 服務搭配使用。 當使用者設定 remote、登入第三方服務、clone、fetch、pull 或 push 時，相關資料會直接在使用者裝置與該第三方服務之間傳輸。

這些第三方服務的資料處理方式，會受到該第三方服務自身的隱私權政策與服務條款約束。 PocketGit 不控制第三方服務如何處理資料。

五、憑證與 Access Token

若使用者選擇使用 GitHub 或其他 Git 服務的 access token、帳號密碼或其他憑證，PocketGit 可能會將這些憑證儲存在使用者裝置本機，用於後續 Git 操作。

PocketGit 不會主動將這些憑證傳送給我們的伺服器。憑證僅會在使用者執行需要驗證的 Git 操作時，用於連線至使用者指定的第三方 Git 服務。

六、Git Server Repo 模式

PocketGit 可能提供 Git Server Repo 模式，讓使用者可以在同一個 Wi-Fi 網路內，從其他裝置 clone 或 push 到 iPhone 上的 repository。

啟用此功能時，使用者的裝置可能會在區域網路內提供 repository 存取。請只在可信任的網路環境中啟用此功能，並在不需要時關閉。

七、分析、廣告與追蹤

PocketGit 不使用廣告 SDK，不會追蹤使用者跨 App 或跨網站的活動，也不會將使用者資料出售給第三方。

若未來加入錯誤回報、分析或其他資料收集功能，我們會更新本隱私權政策，並依照 App Store 規範揭露相關資料使用方式。

八、資料分享

PocketGit 不會將使用者的個人資料分享、販售或出租給第三方。

只有在使用者主動設定第三方 Git remote 並執行 Git 操作時，相關 repository 資料才會傳輸至使用者指定的 Git 服務。

九、資料刪除

使用者可以在 App 內刪除 repository、設定或憑證。使用者也可以透過 iOS 系統刪除 PocketGit App，以移除 App 在裝置上儲存的本機資料。

若資料已經被使用者主動 push 至 GitHub 或其他第三方服務，該資料的刪除需要依照該第三方服務提供的方式處理。

十、兒童隱私

PocketGit 並非專門提供給兒童使用的 App。我們不會故意收集兒童的個人資料。

十一、隱私權政策變更

我們可能會不時更新本隱私權政策。當內容有重大變更時，我們會更新本頁面的最後更新日期。
