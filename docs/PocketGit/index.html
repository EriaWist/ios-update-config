PocketGit Support

在 iPhone 上管理你的 Git Repository

PocketGit 是一款為 iPhone 設計的 Git 工具，協助你瀏覽儲存庫、查看變更、建立 commit、切換 branch，並在行動裝置上處理日常 Git 工作。

管理儲存庫

瀏覽、搜尋與整理 iPhone 上的 Git repositories、資料夾與連結。

Commit 與 Revert

查看變更檔案、選擇要提交的內容、撰寫 commit message，並可選擇是否提交後 push。

Branch 切換

快速查看本地與遠端 branch，並在手機上切換目前工作的分支。

Git Server Repo 模式

PocketGit 支援 Git Server Repo 模式，讓你可以在同一個 Wi-Fi 網路內，從其他裝置 clone 或 push 到 iPhone 上的 repository。

清楚查看 Diff

在提交前檢查新增與刪除內容，降低錯誤提交的機會。

常見問題

PocketGit 需要登入帳號嗎？

PocketGit 可以管理本機 Git repository。若你需要連接 GitHub 或其他遠端 Git 服務，可能需要依照該服務的方式設定憑證或 access token。

我的 repository 會上傳到 PocketGit 的伺服器嗎？

不會。PocketGit 的 repository 與設定主要儲存在你的裝置本機。除非你主動設定 remote 並執行 push，否則不會將內容推送到第三方 Git 服務。

Git Server Repo 模式是什麼？

Git Server Repo 模式可以讓你的 iPhone 在同一個 Wi-Fi 網路內提供 Git repository 連線，方便其他裝置 clone 或 push。
