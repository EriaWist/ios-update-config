Manage Git repositories on your iPhone

PocketGit is a Git repository management tool designed for iPhone. Browse repositories, review changes, create commits, switch branches, and handle everyday Git workflows directly from your mobile device.

Manage Repositories

Browse, search, and organize Git repositories, folders, and linked directories on your iPhone.

Commit and Revert

Review changed files, select what to commit, write commit messages, and optionally push after committing.

Switch Branches

View local and remote branches, then quickly switch to the branch you need.

Git Server Repo Mode

PocketGit supports Git Server Repo Mode, allowing devices on the same Wi-Fi network to clone from or push to a repository hosted on your iPhone.

Review Diffs Clearly

Check added and removed lines before committing, so you can avoid accidental changes.

Frequently Asked Questions

Do I need an account to use PocketGit?

PocketGit can manage local Git repositories without requiring an account. If you connect to GitHub or another remote Git service, you may need to configure credentials or an access token.

Are my repositories uploaded to PocketGit servers?

No. PocketGit stores repositories and settings locally on your device. Repository data is only sent to third-party Git services when you intentionally configure a remote and perform actions such as push, pull, fetch, or clone.

What is Git Server Repo Mode?

Git Server Repo Mode allows your iPhone to provide Git repository access within the same Wi-Fi network, so another device can clone from or push to your iPhone.
