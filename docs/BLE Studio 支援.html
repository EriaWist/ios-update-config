BLE Studio - 支援資訊

針對 BLE Studio 的問題回報、功能建議與技術支援，請參考以下說明。

1. App 簡介

BLE Studio 是一款面向開發者與進階使用者的藍牙低功耗（Bluetooth Low Energy, BLE）工具， 用於掃描、連線、檢視 GATT Service / Characteristic，以及測試讀寫與自動回覆規則。

主要用途包括：

  • 驗證自家裝置或開發板的 BLE 通訊行為
  • 偵錯 GATT Service / Characteristic 的讀寫流程
  • 建立預設指令（Presets）與變數模板，加速測試流程

3. 常見問題（FAQ）

3.1 掃不到我的裝置怎麼辦？

  • 請確認目標裝置有開啟電源並進入廣播模式（Advertising）。
  • 請在 iOS 系統「設定」中確認已允許 BLE Studio 使用藍牙。
  • 如果有勾選「指定 UUID 掃描」，請確認 Service UUID 設定正確，或暫時關閉後再試。

3.2 已連線但讀寫失敗？

  • 請確認選擇的 Characteristic 具備對應權限（Read / Write / Notify）。
  • 確認外部裝置的 GATT 定義與實際行為一致（有些裝置需要先寫入特定命令才能開啟通知）。
  • 可查看 App 內的 Log 區塊，確認實際錯誤訊息。

3.3 自動回覆 / 預設指令無法正常替換變數？

  • 請確認指令格式符合：{{VAR_NAME}}。
  • 在「規則編輯」頁面中檢查 JSON 是否成功解析、變數名稱是否一致。
  • 如有使用正規表達式驗證（regexPattern），請確認輸入內容符合限制。

4. 已知限制

  • 本 App 目前主要針對 BLE / GATT 測試，不支援傳統藍牙（BR/EDR）。
  • 不同廠牌裝置可能對 GATT 規範有私有擴充，實際行為以裝置廠商文件為準。
  • 部分系統版本／裝置可能對背景掃描與連線有額外限制，請以 iOS 官方說明為準。

5. 隱私與資料保護

BLE Studio 主要在您本機裝置上處理藍牙測試相關資料（例如裝置清單、UUID、GATT 內容與測試指令）， 不會主動將這些資料上傳到伺服器。
